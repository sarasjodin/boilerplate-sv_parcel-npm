# Boilerplate

En enkel projektstruktur för frontendutveckling med stöd för Parcel.

## Struktur

- `src/`: Alla källfiler.
- `dist/`: Bundlade filer (skapas automatiskt av Parcel).

## Kom igång

1. Klona projektet.
2. Installera beroenden: `npm install`.
3. Starta utvecklingsservern: `npm start`.
